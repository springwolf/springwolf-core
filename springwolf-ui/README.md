# Springwolf UI
##### Web UI for Springwolf

### Usage
Add the following dependencies:

```groovy
dependencies {
    runtimeOnly 'io.github.springwolf:springwolf-ui:0.3.1'
}
```

After starting the application, visit: `localhost:8080/springwolf/asyncapi-ui.html`.

### Development
1. Run `npm i`
2. Run `ng serve`
